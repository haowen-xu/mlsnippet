Installation
------------

.. code-block:: bash

    pip install git+https://github.com/haowen-xu/mltoolkit.git

    # Optional, for using `mltoolkit.datafs.DataFS.flow` method series
    pip install git+https://github.com/haowen-xu/tfsnippet.git
